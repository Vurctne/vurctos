# Sessions

Session recall is the third memory layer. Each working session is logged here as a dated markdown file, for example `2026-06-28-hook-pass.md`. This is the learning summary made durable and searchable later.

## Session Log Template

```text
# Session: <date> <short title>

## Request
What the user asked for.

## Cards Run
Which board cards ran and which profiles handled them.

## Accepted
What was accepted into the project record.

## Rejected
What was rejected and why.

## Learned
What should be remembered, and any skill candidates.
```

In v1 this is plain markdown. A future version may add a full-text index for cross-session recall, following the Hermes Agent design.
